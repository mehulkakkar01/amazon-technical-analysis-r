# amazon-technical-analysis-r
Technical analysis of Amazon stock using R with SMA, RSI, MACD, and Bollinger Bands to generate trading signals.

# Technical Analysis of Amazon Stock (R)

## Objective
Analyze stock price trends using technical indicators and generate trading signals.

## Tools Used
- R (quantmod, TTR)

## Files in this Repository
- technical_analysis.R → R code for analysis  
- Technical_Analysis_Report.pdf → Detailed explanation and results  

## Key Insights
- SMA shows trend direction  
- RSI identifies overbought/oversold levels  
- MACD confirms momentum  
- Bollinger Bands indicate volatility  

## Conclusion
Using multiple indicators together improves reliability of trading signals
